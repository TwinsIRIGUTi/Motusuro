# ⚡️ THUNDER MOTSU - Thunder V Simulator

1997年の名機「サンダーV」をブラウザ上で再現したスロットシミュレーターです。
スマホ操作に最適化し、実機さながらの「三連V」の快感と「高設定」の爆発力を手軽に体験できます。

## 🌟 主な特徴

- **完全再現のリール配列**: 初代サンダーVの実機データを忠実に再現。
- **左リール「三連V」**: BIG成立時にズバッと止まる「1確」の脳汁演出を搭載。
- **常に設定5 or 6**: 一般公開用に甘めのスペック（設定5・6確定）を実装。
- **技術介入要素**: BIG中のJAC IN外しや小役ゲームの粘りなど、4号機特有の遊び方を再現。
- **スマホ最適化**: タップ操作、バイブレーション演出（1確時）、軽量動作。

## 🕹 遊び方

1. **レバーON**: ボタンをタップしてリールを回転させます。
2. **ストップボタン**: 左・中・右の順にリールを止めます。
3. **ボーナス当選**: 
   - 左リールに**三連V**が止まればBIG確定！
   - 「V・V・V」や「7・7・7」が揃えばBIG BONUS開始。
   - 「BAR・BAR・BAR」が揃えばREG BONUS（100枚獲得）。
4. **BIG BONUS中**:
   - 小役ゲーム（最大30G）でメダルを稼ぎます。
   - リプレイが揃うと「JAC IN」。JACゲーム（8G）で15枚を連続ゲット！
   - 3回のJAC INでボーナス終了となります。

## 📊 スペック（設定5・6混合）

| 項目 | 内容 |
| :--- | :--- |
| **BIG確率** | 約 1/240 〜 1/260 |
| **REG確率** | 約 1/380 〜 1/420 |
| **ベース** | 常に高設定挙動 |
| **1確演出** | 左リール三連V停止時、メッセージ＆振動 |
